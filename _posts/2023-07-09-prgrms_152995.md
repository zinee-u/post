## 문제 정의
> [인사고과](https://school.programmers.co.kr/learn/courses/30/lessons/152995)  

- 입력  
    - 1 ≤ scores의 길이 ≤ 100,000  
    - 평가 점수  
        - a, b = scores[:][0], scores[:][1]   
- 출력  
    - 완호의 석차  

## 해결 방법  
1. 완호 석차가 증가하는 경우   
    - 완호  
2. 완호 석차가 증가하지 않는 경우  
- 내림차순, 오름차순  

```python
def solution(scores):
    scores.sort(key=lambda x: (-x[0], x[1]))
```
