3. 최대, 최소 값 갱신
	1. limit >= p_sum -> 최대, 최소 값 갱신, 보트에 둘을 태울 수 있는 경우
	2. limit < p_sum -> 최대 값 갱신, 단독 보트 필요한 경우
 
```{python}
def solution(people, limit):
    answer = 0 
    people.sort(reverse = True)
    l, r = 0, len(people - 1)
    
    while r >= l:
       p_sum = people[l] + people[r]
       if limit >= p_sum:
            # 보트 필요
            answer += 1
            # 다음 조합
            l += 1
            r -= 1
       else:
            # 단독 보트 필요
            answer += 1
    return answer
```
